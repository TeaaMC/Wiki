# Crystal Royale

Crystal Royale is TeaaMC's battle royale mode — everyone drops in, the storm closes, and only one player or team walks out.

## Joining a match

1. Wait for a host/admin to open the queue with `/queue`.
2. Run `/join` to enter the queue.
3. If you queued by mistake, run `/leave` to back out before the match starts.

::: warning Timing matters
`/join` only works while the queue is open (waiting or countdown). If no queue is open, you'll need to wait for staff to start one.
:::

## Playing as a team

You can team up with others before a match starts:

1. Run `/team` to open the team menu.
2. Click **LIME_STAINED_GLASS_PANE** to create your own team and become its leader.
3. As leader, click an empty member slot and type the player's name in chat to send an invite.
4. The invited player accepts with `/team accept`.

A team can have up to **5 members**. Only the leader can invite or manage members (promote/kick), and teams can only be created or edited while still in the lobby — once the match starts, the roster is locked.

::: tip Not a leader?
If you're a regular member and want to leave your team before the match starts, open `/team`, view your team, and use the leave option.
:::

## During the match

- `/revive [player]` — revive a downed teammate (costs 20 Totems of Undying)
- `/rejoin` — rejoin an in-progress game (or start spectating) if you got disconnected

## Spectating

- `/spectate` (alias `/spec`) — spectate the currently running match
- `/stopspectate` (aliases `/stopspec`) — leave spectator mode and return to the lobby

## Stats

Run `/stats` to view your own stats, or `/stats <player>` to check someone else's.
