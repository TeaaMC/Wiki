# Connect to the server

TeaaMC is a **Practice Crystal PvP** server for Java Edition. You can join with versions **1.20.5 through 26.2**.

![TeaaMC server lobby](/banner2.png)

## Server addresses

| Priority | Address | When to use it |
|---|---|---|
| Primary | `teaamc.asia` | Always try this address first |
| Backup | `103.161.119.175:40001` | When the primary address is temporarily unavailable |

## How to join

1. Open **Minecraft: Java Edition**.
2. Select **Multiplayer** → **Add Server**.
3. Enter `TeaaMC` as the server name and `teaamc.asia` as the address.
4. Save, select TeaaMC and click **Join Server**.

::: warning Version support
TeaaMC currently supports Java Edition only. Use a version within the supported range if you encounter a compatibility error.
:::
