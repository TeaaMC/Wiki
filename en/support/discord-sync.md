# Sync Discord ranks and roles

Link your Minecraft account to Discord so eligible ranks can be synchronized automatically.

## Link your account

Link your account with the `/link` command in game. See the full step-by-step guide (with screenshots) on the [Discord linking & 2FA](/en/support/discord-link) page.

After linking succeeds, leave and rejoin the Minecraft server so your roles sync.

## Synchronized roles

The system checks for updates about once every **3 minutes** for these roles:

`Linked`, `Booster`, `Media`, `Designer`, `Partner`, `Builder`, `Developer`, `Teaplus`, `Helper`, `Mod`, and `Staff`.

::: warning Keep your code private
Never share your `/link` code. If the role does not appear after a few minutes, rejoin the server before opening a support ticket.
:::
