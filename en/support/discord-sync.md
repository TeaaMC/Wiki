# Sync Discord ranks and roles

Link your Minecraft account to Discord so eligible ranks can be synchronized automatically.

## Link your account

1. Join `teaamc.asia`.
2. Run `/discordlink`.
3. Copy the code and follow the instructions returned by the server.
4. Open Discord, go to **⌨┃command**, and use the link command with your code.
5. After linking succeeds, leave and rejoin the Minecraft server.

## Synchronized roles

The system checks for updates about once every **3 minutes** for these roles:

`Linked`, `Booster`, `Media`, `Designer`, `Partner`, `Builder`, `Developer`, `Teaplus`, `Helper`, `Mod`, and `Staff`.

::: warning Keep your code private
Never share your `/discordlink` code. If the role does not appear after a few minutes, rejoin the server before opening a support ticket.
:::
