# TeaaMC Wiki

Tài liệu chính thức song ngữ cho máy chủ Practice CPvP TeaaMC. Tiếng Việt là ngôn ngữ mặc định; bản tiếng Anh nằm dưới `/en/`.

## Phát triển cục bộ

```bash
npm ci
npm run docs:dev
```

## Kiểm tra bản dựng

```bash
npm run docs:build
```

Site được triển khai tự động lên GitHub Pages khi nhánh `main` được cập nhật.
